# 💼 Enterprise Payroll and Tax Processing System

## 📌 Project Overview

The **Enterprise Payroll and Tax Processing System** is a Java-based desktop application developed as a group project.
It is designed to manage employee records, process salaries, calculate taxes, track attendance, and generate payslips efficiently.

This system simulates real-world enterprise payroll operations by integrating tax calculation, structured data management, and document generation.

---

## 🚀 Features

* 👤 Employee Management (Add, Update, View)
* 💰 Salary Management with Automatic Calculation
* 🧾 Tax Processing based on salary slabs
* 📊 Attendance Tracking System
* 📄 Payslip Generation
* 🖨️ **PDF Payslip Export**
* 📋 JTable-based Data Display
* 🔐 Secure Login System

---

## 🧾 PDF Payslip Export

The system allows users to generate and export payslips in **PDF format**, making it suitable for real-world usage.
This feature enhances usability by enabling employees to download and print their salary details.

---

## 🧮 Tax Processing

The system calculates tax based on predefined salary slabs and computes net salary after deductions, simulating real-world enterprise payroll systems.

---

## 🛠️ Technologies Used

* Java (Swing, AWT)
* MySQL Database
* JDBC (Java Database Connectivity)
* NetBeans IDE
* PDF Generation Library (e.g., iText / ReportLab equivalent in Java)

---

## 📂 Modules

* Login System
* Employee Management
* Salary Management
* Attendance Management
* Payslip Generator
* PDF Export System
* Data Visualization (JTable)

---

## 🏗️ System Workflow

1. User logs into the system
2. Employee details are added or updated
3. Salary is calculated with tax deductions
4. Attendance is recorded
5. Payslip is generated
6. Payslip can be exported as PDF
7. Data can be viewed in tabular format

---

## ▶️ How to Run

1. Install MySQL and create the required database
2. Import the project into NetBeans IDE
3. Add MySQL JDBC Driver (Connector/J)
4. Add required PDF library (if applicable)
5. Run `login_page.java`
